No, This Theme is only for your commercial use.

In case, any bug raised in our listed features of this theme, we're committed to provide free support. We provide paid support for special requirements like Theme Installation, Configuration & Customization as per your need.

You can definitely shoot an email at odoo@keyconcepts.co.in and we will be happy to make it compatible with your Odoo version with extra efforts.

This module includes 1 months of support period.

6) Do you install theme for us?

Yes, sure you can purchase the theme with installation service and we would be happy to install it for you.

7) Is this theme responsive?

Yes, it is responsive.

8) Can you personalize theme for me?

Yes, of course you can share your requirements at odoo@keyconcepts.co.in and we’ll get back to you with the proceedings.

9) Will I be able to add my own CSS to the theme?

Yes, you can create new modules and extend theme CSS. Please do not alter the theme CSS directly as it may break the theme design.
